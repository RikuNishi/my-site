# my-site

## 概要
このリポジトリでは、Appleのプロモーションサイトを参考にした、りくにしさんの自己紹介ページを1つの静的HTMLで構成しています。洗練されたヒーロー、セクション、ギャラリーを通じて、プロフィール、活動領域、趣味をわかりやすく伝える設計です。

## ローカル確認
- `index.html` を直接ブラウザで開くと最小限の環境でも表示確認できます。
- `python -m http.server` などの軽量サーバーをプロジェクトルートで起動すれば `http://localhost:8000` から動作をチェックできます。

## ディレクトリ構成
- `index.html`: ヒーロー部、プロフィール・フォーカス・趣味セクション、フッターまで含む単一ページ。
- `styles.css`: ガラスっぽいカード、タイポグラフィ、レスポンシブなグリッドなどのスタイル全般。
- `turtle_img.jpg`: 趣味セクションで使用する画像資産。
- `AGENTS.md`: 貢献者向けガイド。ワークフローの変更を反映してください。
- `particles.js`: 以前のp5.js実験用スケッチ。現在ページに読み込まれていないため不要なら削除して構いません。

## デプロイ
GitHub Pagesを想定した構成です。`main` ブランチ（またはデフォルトブランチ）にコミットして、Pages 設定でルート (`/`) から公開するようにしてください。

## 備考
- ビルドツールや依存関係は使っていないので、変更は軽量な静的ファイルの更新に留めるのが望ましいです。
- 新たなビルド・テスト・CIルールを導入した際は `AGENTS.md` もあわせて更新し、協力者に周知しましょう。
